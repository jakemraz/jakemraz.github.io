# 안녕하세요 👋

저는 **개발과 글쓰기**를 사랑하는 사람입니다.

## 🚀 무엇을 하나요?

이 블로그는 제가 배우고, 경험하고, 생각한 것들을 기록하는 공간입니다.  
주로 다음과 같은 주제들을 다룹니다:

- **개발**: 웹 개발, 프로그래밍 언어, 새로운 기술
- **책쓰기**: 글쓰기 여정과 배움
- **일상**: 생각의 조각들과 경험담

## 💡 왜 블로그를 쓰나요?

> "글을 쓴다는 것은 생각을 정리하는 가장 좋은 방법이다."

배운 것을 정리하고, 생각을 명확하게 하며, 다른 사람들과 공유하기 위해 글을 씁니다.  
때로는 과거의 나에게, 때로는 미래의 누군가에게 도움이 되기를 바라며 기록합니다.
